ОПИСАНИЕ 

АВТОР
Кубышкин Иван Олегович

Создал GUI-приложение «Currency Converter» с использованием внешнего API, сохранением истории и Git.История конвертаций может сохраняться в файл (например, в формате JSON), что позволяет сохранять данные между запусками. Система контроля версий (Git) используется для отслеживания изменений в коде, совместной работы над проектом, возврата к предыдущим версиям и других задач. приложения.
