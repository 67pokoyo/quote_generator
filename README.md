# 🎲 Random Quote Generator

**Автор:** Кубышкин Иван Олегович

## 📝 Описание

Приложение для генерации случайных цитат с возможностью фильтрации по автору и теме. 
Программа сохраняет историю всех сгенерированных цитат в JSON файл и позволяет добавлять собственные цитаты.

## ✨ Функциональность

- ✅ Генерация случайной цитаты из предопределённого списка
- ✅ Фильтрация цитат по автору и теме
- ✅ Добавление новых цитат пользователем (с валидацией)
- ✅ История всех сгенерированных цитат с временными метками
- ✅ Сохранение истории и пользовательских цитат в JSON
- ✅ Экспорт истории в отдельный JSON файл
- ✅ Очистка истории с подтверждением
- ✅ Просмотр выбранной цитаты из истории
- ✅ Предотвращение добавления дубликатов

## 🚀 Требования

- Python 3.7+
- Стандартные библиотеки (tkinter, json, random, datetime, unittest)
